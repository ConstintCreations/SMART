# SMART
SMART is an interactive goal management app that helps users keep track of their goals with the SMART goal framework.
## Features
SMART features:
 - Goal Creator: Helps users easily create meaningful and detailed goals through the goal creation quiz.
 - Progress Tracking: Each goal has a progress bar attached to it, showing how close the goal is to completion.
 - Searching & Filtering: Using the search bar and filter, goals can be quickly found and sorted.
 - Goal Editing: Goals can be easily edited both before and after creation.
 - Goal Completion: Marking goals as complete are celebrated with confetti!
